- [T-014] `eval/pu_metrics.py`: the SCAR-safe reporting metrics (plan.md ┬¦7; RevClassify `final_test/{f1,prauc}`). **Four pure-numpy fns + `pu_metric_report` bundler**, label convention `1`=suspicious/positive `0`=licit/unlabeled (shared w/ Stage 2). **`pr_auc(scores,labels)`** = average precision `╬Ż ╬öR┬ĘP` the sklearn way (NOT trapezoid): sort desc by score (mergesort, stable ties), `tps/fps=cumsum`, collapse to one point per distinct score via `boundaries=r_[where(diff(s))[0], n-1]`, `precision=tps/(tps+fps)`, `recall=tps/n_pos`, anchor at `(R=0,P=1)` then `sum(diff(recall)*precision[1:])`. - [T-011] `pu/encoder.py`: heterophily-tolerant GNN encoder (plan.md ┬¦2/┬¦3). **Core layer** `EgoNeighborConv(MessagePassing, aggr="mean")`: `h_v' = W_ego┬Ęx_v + W_neigh┬ĘAGG_{uŌłłN(v)} x_u` with TWO independent linear maps and **NO self-loops** ŌĆö the ego path (`lin_ego`, with bias) bypasses aggregation entirely so a node's own features are preserved against a class-mismatched (heterophilous) or empty neighborhood.
