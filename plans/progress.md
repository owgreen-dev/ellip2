- [T-010] `pu/prior_estimation.py`: TIcE class-prior estimator (Bekker & Davis 2018, AAAI) in the **SCAR** framing + `prior_sweep_grid`. **Framing:** single dataset with a boolean `labeled` mask (`s=1` = labeled positive, `s=0` = unlabeled); estimates `c=P(s=1|y=1)` then `π̂_p = γ/ĉ` where `γ=P(s=1)`. **TIcE primitive:** within ANY subdomain T the labeled fraction `#lab_T/#tot_T ≤ #lab_T/#pos_T = c` (since `#pos_T ≤ #tot_T`), so every subdomain's labeled fraction is a LOWER bound on c; `ĉ = max_T (labeled_frac − correction)`, tightest on a (near-)pure-positive region. **Subdomain induction** `_subdomain_masks`: small greedy decision tree — each node splits on the single feature/median-threshold whose child is richest in labeled examples; both children become candidates + recursed to `max_depth` (default 4); full domain always a candidate. **Key bound used twice:** `c ≥ γ` always (full sample is a valid subdomain) → floor `ĉ` at γ, which guarantees `π̂_p = γ/ĉ ∈ (0,1]` (no upper clamp needed, naturally ≤1). **delta** (default 0.0 = no correction) applies a Hoeffding downward correction `sqrt(ln(1/δ)/(2|T|))` to each subdomain's frac → smaller ĉ → LARGER (more conservative) π̂_p; assert `conservative.prior ≥ plain.prior`. `_C_FLOOR=1e-6` guards the div. **`prior_sweep_grid(lo=1e-5,hi=1e-1,num=5)`** = `np.geomspace` (one point per decade by default → [1e-5..1e-1]), `include=` merges a point (e.g. the estimate) via `np.unique`; validates 0<lo<hi<1, num≥2. plan.md §2: π_p is the small CLUSTER-level prior, NOT the 2.27% subgraph rate; heterophily biases it up so we SWEEP both directions. **Test design:** separable SCAR mixture pos~N(+5,1)/neg~N(−5,1), seeded RNG → a `feature>0`-ish subdomain is ~pure-positive so labeled_frac there ≈ c; π=0.25,c=0.6 recovered within abs 0.06 (γ≈0.15). All pure-numpy, `npt.NDArray` typing. 104 tests green (6 new), ruff+mypy clean.
- [T-009] `pu/nnpu_loss.py` (new `pu/` pkg, first PU module): non-negative PU risk (Kiryo 2017, arXiv:1703.00593). **Surrogate** `sigmoid_loss(logits, positive: bool)` = σ(−t·z): `positive=True`→σ(−z)=ℓ(·,+1), `False`→σ(z)=ℓ(·,−1); symmetry ℓ⁺+ℓ⁻=1 (asserted) is why sigmoid is the recommended PU surrogate. **Core** `_risk_parts(p_logits, u_logits, prior, *, surrogate, beta)` → frozen `PURiskParts(positive_risk=π_p·E_P[ℓ⁺], negative_risk=E_U[ℓ⁻]−π_p·E_P[ℓ⁻] (UNBIASED, pre-clamp), clamped: bool)`. Public: `nnpu_risk(...)` returns `positive_risk + clamp(negative_risk, min=−beta)` as a 0-dim grad tensor (default beta=0 → plain non-negativity; `return_parts=True` → `(risk, parts)`); `upu_risk(...)` = unbiased (no clamp) for comparison; `nnpu_risk_for_backward(..., gamma=1.0)` = Kiryo training rule (when clamped, returns `−γ·negative_risk` so backward ASCENDS the over-fit term; else == unbiased). **Key facts:** π_p is the small CLUSTER-level prior, NOT the 2.27% subgraph rate (Resolved decision #2). `prior∈(0,1)` and non-empty P & U enforced (ValueError). `surrogate: SurrogateLoss = Callable[[Tensor,bool],Tensor]` is injectable. **Test design for the clamp:** to force `negative_risk<0`, make E_U[ℓ⁻] small (U logits strongly negative, σ(z)≈0) AND π_p·E_P[ℓ⁻] large (P logits strongly positive, σ(z)≈1) with π_p=0.5 → clamp engages, risk==positive_risk, and upu_risk dips strictly below. Hand-value test uses the well-separated regime (negative term >0) so nnPU==unbiased and a pure-Python σ re-derivation matches. **Gotchas:** ruff UP035 → import `Callable` from `collections.abc` not `typing`; torch already in mypy override list (ignore_missing_imports) so no stub work. 99 tests green (7 new), ruff+mypy clean.

- [T-005] `features/temporal.py`: `compute_temporal_features(edge_index, timestamps, n_nodes, *, empty_value=0.0)` → dict keyed by `COLUMNS = ("activity_span", "burstiness")`, both float64 (N,). **Events = incident edges** (direction-agnostic): each edge is one event for BOTH endpoints via `node=concat(src,dst)`, `event_ts=concat(ts,ts)` — so incident-event count == `total_degree`, and a self-loop yields two same-ts events. `timestamps` is the 2nd POSITIONAL arg (like `weights` in flow_concentration), length E. **activity_span** = per-node `max−min` via `np.minimum.at`/`np.maximum.at` (init ±inf); genuine `0.0` for a single event, `empty_value` only when zero events. **burstiness** = Goh&Barabási `B=(σ−μ)/(σ+μ)` over inter-event GAPS, B∈[−1,1] (−1 regular/periodic, 0 Poisson, →1 bursty). Vectorised gaps: `order=lexsort((event_ts,node))`; gap between consecutive same-node sorted events (`sn[1:]==sn[:-1]`) belongs to `sn[1:]`, value `st[1:]-st[:-1]` (≥0). **Population** std (matches edge_aggs): `var=Σg²/n − μ²`, clip≥0 for fp. **empty_value** for burstiness when <2 events (no gap) OR all events share one ts (`σ+μ==0`, denom 0). **Clean hand values for tests:** equal gaps→−1; single gap (2 events)→−1; gaps[0,2]→0; gaps[1,9] (μ5,σ4)→−1/9; k zeros + one spike → `(√k−1)/(√k+1)` (k=4 → 1/3, a clean bursty>0). Test isolates each measured source node (0..7) with its own out-edges to dedicated sinks (idx≥10) so incident sets are controlled; node 7 mixes one in + one out edge to assert incident = in∪out. 14 tests, gate green (65 total, ruff+mypy clean).
