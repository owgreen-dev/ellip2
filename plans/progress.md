- [T-016] `exit_paths/path_search.py` was already present (untracked, complete); this iteration added the missing `tests/test_path_search.py` (plan.md decision #4 + §3). **Module API (for downstream T-017):** `reachability(edge_index, sources, endpoints, n_nodes, *, max_hops=MAX_HOPS(=6), frontier_cap=None, hubs=None) -> ReachabilityResult`; lower-level `bfs_reachable(adjacency, seeds, *, max_hops, frontier_cap, hub_mask) -> BFSResult`; `induced_subgraph(edge_index, node_mask) -> (sub_ei, node_ids)`. **Semantics asserted by the test (the 4 acceptance properties):** (1) **reachability within k** — a candidate reaches the endpoint set iff its BACKWARD distance ≤ max_hops; the backward sweep runs on the *transposed* adjacency from endpoints, so `backward.hops[v]` = #hops `v→…→endpoint` (chain 0→1→2→3, endpoint 3 ⇒ back.hops=[3,2,1,0]); max_hops=2 makes the same candidate `candidate_reaches=False` (k is a hard bound). - [T-014] `eval/pu_metrics.py`: the SCAR-safe reporting metrics (plan.md §7; RevClassify `final_test/{f1,prauc}`). **Four pure-numpy fns + `pu_metric_report` bundler**, label convention `1`=suspicious/positive `0`=licit/unlabeled (shared w/ Stage 2). **`pr_auc(scores,labels)`** = average precision `Σ ΔR·P` the sklearn way (NOT trapezoid): sort desc by score (mergesort, stable ties), `tps/fps=cumsum`, collapse to one point per distinct score via `boundaries=r_[where(diff(s))[0], n-1]`, `precision=tps/(tps+fps)`, `recall=tps/n_pos`, anchor at `(R=0,P=1)` then `sum(diff(recall)*precision[1:])`. It's a LOWER BOUND under SCAR (hidden positives in the unlabeled pool depress apparent precision). **`binary_f1`**=`2TP/(2TP+FP+FN)`, **`recall`**=`TP/(TP+FN)` (the one thing SCAR estimates unbiasedly), **`lee_liu_score`**=`recall²/Pr(ŷ=1)` (Lee&Liu 2003 lift — rewards recall, penalises predicting-positive-indiscriminately via the over-ALL-data denom). Threshold-dependent three take **binary preds**; only pr_auc takes raw scores. `pu_metric_report(scores,labels,*,threshold=0.5)` thresholds `scores>=threshold` → dict keyed `COLUMNS=("prauc","f1","recall","lee_liu")`. **Degenerate guards (return 0.0, don't raise):** f1 when `2TP+FP+FN==0`, recall when no positives, lee_liu when no predicted positives; pr_auc DOES raise on zero positives or length mismatch. **`_as_binary`** rejects non-0/1 (`array_equal` after int64 cast catches floats like 0.5). **Clean hand values for tests:** scores`[.9,.8,.7,.6]`/labels`[1,0,1,0]`→AP=**5/6**; yt`[1,0,1,0,1]`/yp`[1,0,1,1,0]` (TP2,FP1,FN1)→f1=recall=**2/3**, lee_liu=`(4/9)/(3/5)`=**20/27**; selective`[1,1,0,0]`→2.0 > indiscriminate`[1,1,1,1]`→1.0 on yt`[1,1,0,0]` (proves the over-prediction penalty); tied `[.5,.5,.5,.5]`→0.5 (single boundary). - [T-011] `pu/encoder.py`: heterophily-tolerant GNN encoder (plan.md §2/§3). **Core layer** `EgoNeighborConv(MessagePassing, aggr="mean")`: `h_v' = W_ego·x_v + W_neigh·AGG_{u∈N(v)} x_u` with TWO independent linear maps and **NO self-loops** — the ego path (`lin_ego`, with bias) bypasses aggregation entirely so a node's own features are preserved against a class-mismatched (heterophilous) or empty neighborhood.
