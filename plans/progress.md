- [T-011] `pu/encoder.py`: heterophily-tolerant GNN encoder (plan.md В§2/В§3). **Core layer** `EgoNeighborConv(MessagePassing, aggr="mean")`: `h_v' = W_egoВ·x_v + W_neighВ·AGG_{uвҲҲN(v)} x_u` with TWO independent linear maps and **NO self-loops** вҖ” the ego path (`lin_ego`, with bias) bypasses aggregation entirely so a node's own features are preserved against a class-mismatched (heterophilous) or empty neighborhood. This is the GraphSAGE/H2GCN "never mix ego into neighbor agg" fix vs pure-GCN smoothing (which folds self-loops into one symmetric-normalized mean). `lin_neigh` is **bias-free** on purpose вҶ’ an isolated node (no in-neighbors) gets `propagateвҶ’0` so its neighbor contribution is EXACTLY 0 and output == `lin_ego(x_iso)` (the key testable property). `message(x_j)=x_j` (source-node features; PyG flow source_to_target, edge col `[s,t]`=`sвҶ’t` so `t` aggregates from in-neighbor `s`). **Encoder** `HeterophilyEncoder(in,hidden,out=None,*,num_layers=2,aggr,dropout,activation=relu,normalize,bias)`: stacks layers, activation+dropout after every layer EXCEPT the last (last stays linear so the PU head sees raw embeddings), optional final L2 `F.normalize`. `dims=[in]+[hidden]*(num_layers-1)+[out]`; `num_layers=1` вҶ’ single `inвҶ’out` map (out defaults to hidden). `num_layers<1` raises ValueError. **mypy gotcha:** iterating `nn.ModuleList` yields `Module` whose `__getattr__` is typed `Tensor|Module`, so `conv.reset_parameters()` errors "Tensor not callable" вҖ” `cast(EgoNeighborConv, conv).reset_parameters()` fixes it (calling `conv(...)` via `__call__` is fine, no cast needed). **Test design:** tiny 6-node ring + 1 deliberately ISOLATED node (idx 5); asserts output shape `(N,out)`, `loss.backward()` populates finite grads on EVERY param (no dead paths), isolated-node embedding == `lin_ego(x_iso)` and changes iff its OWN features change (and can't affect others вҖ” no edges), zeroing `lin_neigh.weight` вҶ’ pure per-node ego transform (proves the two paths are separate), and `normalize=True` вҶ’ unit-norm rows. - [T-010] `pu/prior_estimation.py`: TIcE class-prior estimator (Bekker & Davis 2018, AAAI) in the **SCAR** framing + `prior_sweep_grid`. **Framing:** single dataset with a boolean `labeled` mask (`s=1` = labeled positive, `s=0` = unlabeled); estimates `c=P(s=1|y=1)` then `ПҖМӮ_p = Оі/Дү` where `Оі=P(s=1)`. **TIcE primitive:** within ANY subdomain T the labeled fraction `#lab_T/#tot_T вүӨ #lab_T/#pos_T = c` (since `#pos_T вүӨ #tot_T`), so every subdomain's labeled fraction is a LOWER bound on c; `Дү = max_T (labeled_frac вҲ’ correction)`, tightest on a (near-)pure-positive region. **Subdomain induction** `_subdomain_masks`: small greedy decision tree вҖ” each node splits on the single feature/median-threshold whose child is richest in labeled examples; both children become candidates + recursed to `max_depth` (default 4); full domain always a candidate. **Key bound used twice:** `c вүҘ Оі` always (full sample is a valid subdomain) вҶ’ floor `Дү` at Оі, which guarantees `ПҖМӮ_p = Оі/Дү вҲҲ (0,1]` (no upper clamp needed, naturally вүӨ1). **delta** (default 0.0 = no correction) applies a Hoeffding downward correction `sqrt(ln(1/Оҙ)/(2|T|))` to each subdomain's frac вҶ’ smaller Дү вҶ’ LARGER (more conservative) ПҖМӮ_p; assert `conservative.prior вүҘ plain.prior`. `_C_FLOOR=1e-6` guards the div. **`prior_sweep_grid(lo=1e-5,hi=1e-1,num=5)`** = `np.geomspace` (one point per decade by default вҶ’ [1e-5..1e-1]), `include=` merges a point (e.g. the estimate) via `np.unique`; validates 0<lo<hi<1, numвүҘ2. plan.md В§2: ПҖ_p is the small CLUSTER-level prior, NOT the 2.27% subgraph rate; heterophily biases it up so we SWEEP both directions. **Test design:** separable SCAR mixture pos~N(+5,1)/neg~N(вҲ’5,1), seeded RNG вҶ’ a `feature>0`-ish subdomain is ~pure-positive so labeled_frac there вүҲ c; ПҖ=0.25,c=0.6 recovered within abs 0.06 (ОівүҲ0.15).
