# Data

This repository ships **code only**. The Elliptic2 dataset is **NOT redistributed** here —
you must download it yourself from the source below.

## Source

- **Kaggle:** `ellipticco/elliptic2-data-set`
- **Paper:** *"The Shape of Money Laundering: Subgraph Representation Learning on the
  Blockchain with the Elliptic2 Dataset"* (arXiv:2404.19109).

## Size

- **~24 GB compressed / ~83 GB extracted** (5 CSVs).

## Contents (paper Table 1)

- **49,299,864** background clusters (nodes)
- **196,215,606** background edges
- **121,810** labeled subgraphs — **2,763 suspicious / 119,047 licit** (base rate 2.27%)
- 43 node features, 96 edge feature columns in the real CSV.
